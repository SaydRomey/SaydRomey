
# ASCII Art

## ASCII Art Centering

### `<pre>`

Why Use `<pre>`?
  - Maintains exact formatting (spaces, new lines, indentation).
  - Keeps ASCII art properly aligned inside a `<div>`.
  - Ensures GitHub README renders ASCII art exactly as written.
